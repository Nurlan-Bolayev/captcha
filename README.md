# Captcha

1. cd folder
2. git clone https://github.com/Nurlan-Bolayev/captcha.git
3. Type `code .` (open in vscode)
4. Press `ctrl + '` to open terminal
5. Type `powershell`
6. Type `./cap_env/Scripts/activate`
7. pip install Flask
8. pip freeze > requirements.txt
9. pip install -r requirements.txt
10. Type `$env:FLASK_APP = 'ex'`
11. Type `flask run`
   (this will start dev server)
12. Open in browser (http://localhost:5000)
